# CarBeep

Telegram-бот для мониторинга доступных машин Делимобиля. Следит за заданными зонами и присылает уведомление с диплинком на бронирование, как только рядом появляется свободная машина.

## Возможности

- Создание зон наблюдения по геолокации с выбором радиуса
- Уведомления о новых машинах в зонах (поллинг каждые 15 сек)
- Диплинк на конкретную машину в приложении Делимобиль (`delimobil://map/car/{госномер}`)
- Краудсорсинг госномеров — пользователи сообщают номера, бот запоминает
- Фильтр по моделям, расписание зон, тихие часы
- 12 городов присутствия Делимобиля

## Быстрый старт

```bash
cp .env.example .env
# Заполни CARBEEP_BOT_TOKEN (токен от @BotFather)
nano .env

docker compose up --build
```

## Конфигурация

Все переменные с префиксом `CARBEEP_`:

| Переменная | По умолчанию | Описание |
|---|---|---|
| `CARBEEP_BOT_TOKEN` | — | Токен Telegram бота (обязательно) |
| `CARBEEP_REGION_ID` | `16` | Регион по умолчанию (16 = Ярославль) |
| `CARBEEP_POLL_INTERVAL` | `15` | Интервал опроса API (секунды) |
| `CARBEEP_POSTGRES_HOST` | `postgres` | Хост PostgreSQL |
| `CARBEEP_POSTGRES_PORT` | `5432` | Порт PostgreSQL |
| `CARBEEP_POSTGRES_DB` | `carbeep` | Имя базы данных |
| `CARBEEP_POSTGRES_USER` | `carbeep` | Пользователь БД |
| `CARBEEP_POSTGRES_PASSWORD` | `carbeep` | Пароль БД |
| `CARBEEP_REDIS_URL` | `redis://redis:6379/0` | URL Redis |

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список команд |
| `/now` | Машины в зонах прямо сейчас |
| `/nearest` | Ближайшая машина к каждой зоне |
| `/zones` | Список зон (вкл/выкл, удалить, переименовать) |
| `/home` / `/work` | Быстрое создание именованных зон |
| `/filter` | Фильтр по моделям (kia, rio, polo...) |
| `/schedule` | Расписание мониторинга зон |
| `/settings` | Уведомления, тихие часы |
| `/region` | Сменить город |
| `/plate` | Как сообщить госномер машины |
| `/status` | Статистика |
| `/stop` | Выключить уведомления |

## Диплинки

Формат: `delimobil://map/car/{госномер}` (например `delimobil://map/car/P074PC76`).

Госномера не доступны через публичный API Делимобиля. Решение — краудсорсинг: пользователь видит машину в приложении, сообщает госномер боту через кнопку «📝 Сообщить госномер». Бот запоминает связку `car_id → госномер`. Со второго раза уведомление приходит с кнопкой прямого бронирования.

## Логи

Логи пишутся в `./logs/` (маунтится через docker volume). Ротация по 10 MB, хранение 7 дней, сжатие gz.
